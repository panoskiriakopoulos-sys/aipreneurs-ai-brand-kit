# Aipreneurs AI Brand Kit

**Generate ready-to-paste ChatGPT prompts for professional brand assets.**
Part of the [Aipreneurs Academy] course library.

---

## For Claude Users

1. Open [Claude.ai](https://claude.ai) -> Projects -> Project Settings
2. Copy-paste `SKILL.md` into Custom Instructions
3. Start chatting

Or paste `SKILL.md` at the start of any conversation.

## What It Does

| Asset | ChatGPT Produces |
|-------|-----------------|
| Logo (5 styles) | Transparent PNG, vector style |
| Color Palette | Hex codes + usage guide |
| Brand Board | Visual identity slide |
| Hero Image(s) | Ultra-realistic photos |
| Business Card | Flat mockup design |
| Social Kit | Profile pics, covers, banners |
| Post Templates | Clean social layouts |
| Highlight Covers | 6 icon set |
| Phone Mockup | Realistic device mockup |

## How It Works

1. Tell Claude your client/business name
2. Claude asks a few questions (vibe, colors, location)
3. Claude outputs a ready-to-paste ChatGPT prompt in a code block
4. Copy -> Paste into ChatGPT -> Done

Supports Greek and English. Covers everything from a quick logo to a full brand identity kit.
