# Aipreneurs AI Brand Kit

**Generate professional ChatGPT prompts for brand assets -- logos, colors, hero images, social media, and more.**

Part of the [Aipreneurs Academy] course library.

---

## How to Install

### Option 1: Claude Project (Recommended)

1. Open your [Claude.ai](https://claude.ai) dashboard
2. Go to **Projects** and create a new project or open an existing one
3. Click **Project Settings**
4. Paste the contents of `SKILL.md` into the **Custom Instructions** field
5. Save
6. Start a new conversation in that project

### Option 2: Claude Desktop App

1. Open Claude Desktop
2. Go to **Settings > Projects** (or create a new project)
3. Paste `SKILL.md` contents into the Custom Instructions
4. Start chatting

### Option 3: Direct Conversation

Just paste `SKILL.md` content at the start of any Claude conversation as context.

---

## How to Use

1. **Tell Claude about your client:**
   - "I need a logo for a beachside taverna in Crete called To Kyma"
   - "Help me with branding for a web agency called Pixelcraft"
   - "I need a hero image for a spa in Santorini"

2. **Claude asks** a few questions one at a time (industry, vibe, colors, location).

3. **Claude outputs** a ready-to-paste ChatGPT prompt in a code block.

4. **Copy it and paste it into ChatGPT** (free version works fine).

5. **Done.** Repeat for other assets.

## Tips for Best Results

- **Logos:** Always ask Claude to generate the prompt, paste into ChatGPT. If the logo has a white background, tell ChatGPT "Remove the background, transparent PNG please."
- **Text fix:** ChatGPT sometimes misspells brand names in logos. If that happens, generate the icon alone and add text in Canva (free).
- **Color palettes:** Run the palette prompt first, then tell Claude the hex codes for other assets.
- **Hero images:** Use "ultra-photorealistic 8K" for best quality. Add "golden hour" or "soft daylight" for natural lighting.
